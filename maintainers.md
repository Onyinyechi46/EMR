# Support

## Where to Ask Questions

- **GitHub Discussions**: Preferred for Q&A, feature ideas, and general questions
- **GitHub Issues**: For bug reports and actionable work items
- **Email**: ramambeda@gmail.com (for sensitive/private matters)

## Response Expectations

- This is an open-source educational project maintained on a best-effort basis
- We aim to respond within 2-3 business days
- Community contributions and discussions are encouraged
- PRs are always welcome for fixes and improvements

## What to Include in Bug Reports

### Environment Information
- **OS**: Windows/Linux/macOS version
- **Browser**: Chrome/Firefox/Edge version
- **PHP version**: `php --version`
- **MySQL version**: `mysql --version`
- **Wallet**: Lace wallet version
- **Node version**: `node --version` (if applicable)

### Application Information
- **EMR Pro version**: From CHANGELOG or footer
- **What you were doing**: Steps to reproduce
- **Expected behavior**: What should happen
- **Actual behavior**: What actually happened
- **Error messages**: Full error text or screenshots
- **Transaction hashes**: If applicable

## Common Issues and Solutions

| Issue | Likely Solution |
|-------|----------------|
| Wallet won't connect | Ensure Lace wallet is installed and unlocked |
| Transaction fails | Check you have enough ADA (3 ADA minimum) |
| Records not loading | Refresh UTXOs or reconnect wallet |
| Appointment not showing | Check localStorage or refresh dashboard |
| Registration fails | Verify email format and required fields |
